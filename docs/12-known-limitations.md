# Known Limitations

This is an initial scaffold, not a polished final product.

## Current limitations

- The Gantt uses broad strategic time anchors rather than exact dates.
- Dependency markers are shown as badges / tooltips, not full dependency arrows.
- The app is currently a single React file rather than fully decomposed into components.
- There is no deployment workflow yet.
- There is no automated JSON schema validation yet.
- Status tracking is not yet implemented.

## Likely next improvements

- Add GitHub Pages deployment workflow.
- Add dependency highlighting on click.
- Add a JSON schema for plan data.
- Add status fields and filters.
- Split React components into separate files.
- Add print / export styling.
