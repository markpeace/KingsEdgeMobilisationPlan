## Notes for detailed pages

Each deliverable page should expand the relevant cell into a full plan.

It should include:

- problem solved
- what changes
- components
- delivery steps
- dependencies
- feeds into
- related deliverables

The summary in the overview should remain short. The detail should live on click-through pages.
